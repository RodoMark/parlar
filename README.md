# speak
Video and text chat app that makes heavy use of WebsocketRTC and React-three-fiber.

### Front-End: 
React, React Three Fiber, ThreeJS, SocketIO, SimplePeerJS, 
### Back-End: 
Express, NodeJS, SocketIO
### Database: 
PostgreSQL
